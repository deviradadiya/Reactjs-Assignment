What are React hooks? How do useState() and useEffect() hooks work in functional components?

React Hooks

React hooks are functions that allow you to use state and lifecycle features in functional components.

  • useState()
  • Manages state in functional components.
  • Returns an array: the current state value and a function to update it.
  • useEffect()
  • Handles side effects (e.g., data fetching).
  • Runs after render and can return cleanup functions.
